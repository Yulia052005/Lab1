# Лабораторная работа №1
## Прикладное программирование

### Тема: 
ООП в Python. Обработка исключительных ситуаций. Форматы XML и JSON

### Описание:
Разработана объектно-ориентированная модель транспортной системы с обработкой исключений и работой с форматами данных.

### Файлы:
- `transport_system.py` - основной код программы
- `transport_data.json` - пример данных в JSON формате
- `transport_data.xml` - пример данных в XML формате
